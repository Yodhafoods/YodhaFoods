#flow of the upload an image to Cloudinary

ADMIN → selects image
        ↓
Frontend uploads to Cloudinary (Preset)
        ↓
Cloudinary returns {url, public_id}
        ↓
Frontend sends product JSON with images[] to backend
        ↓
Backend saves images[] in MongoDB
        ↓
Frontend loads product using images[].url
        ↓
CDN delivers optimized image



Payment flow

DB Order (PLACED)
      ↓
Create Razorpay Order (no money)
      ↓
Frontend opens Razorpay UI
      ↓
User pays
      ↓
Frontend receives payment details
      ↓
Backend verifies signature
      ↓
Order = PAID + CONFIRMED
      ↓
Stock deducted
